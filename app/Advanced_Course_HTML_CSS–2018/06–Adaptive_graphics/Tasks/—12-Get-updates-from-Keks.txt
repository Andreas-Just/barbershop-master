Кекс снова подготовил для вас сюрприз: ещё один блок с графикой, на этот раз с ретиновой и векторной. Всё это он закоммитит в ваш мастер-репозиторий после того, как вы нажмёте зелёную кнопку «Получить обновления».

Коммиты с обновлениями придут в мастер-репозиторий, забрать их в свой форк вам нужно самостоятельно. Для этого выполните следующие команды в консоли:

// Все команды набираются в локальной папке репозитория-форка
git checkout master

// Мастер-репозиторий (academy) должен быть настроен, вы делали это раньше
git pull academy master

// Не забудьте отправить локальные изменения в Гитхаб
git push origin master
