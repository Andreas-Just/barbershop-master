После обновления от Кекса попробуйте запустить линтеры локально. Для этого:

1. Установите Node.js [https://nodejs.org/en/].

2. В консоли в папке проекта запустите команду npm i, чтобы установились необходимые зависимости.

3. После установки введите команду npm test и убедитесь, что консоль не выдаёт ошибок.

4. Результаты проверки можно обсудить с наставником.
